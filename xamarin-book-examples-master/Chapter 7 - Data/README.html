# Chapter 7 - Data Access Using SQLite and Data Binding
SQLite is the database of choice for many Xamarin developers. Store and retrieve data locally by using SQLite.NET or ADO.NET. Using Xamarin.Forms data binding, fuse UI elements to your data models. Use the MVVM pattern by binding to a view model. These are the solutions in this chapter and their platforms:
  • ADOdotNETexample                <- Xamarin.Forms C# with cross-platform data access code
  • DataBindingExamples.Xaml     <- Xamarin.Forms XAML
  • DataBindingExamples              <- Xamarin.Forms C#
  • SQLiteNETSharedProject         <- Xamarin.Forms C# with cross-platform data access code
  • SQLiteNetPCL                          <- Xamarin.Forms C# with cross-platform data access code
